# 四系设计准则

本模组现在把历史上的 `nihility_*` 注册名视为兼容性前缀，不再把所有内容都硬叫“虚无”。玩家可见名称、提示、贴图、粒子、功能、配方应按下面四系分工处理。

## 虚无 / 无相

- 核心概念：绝对空无、归零、抹除，连“虚空空间”也不存在。
- 视觉语言：黑白灰、缺失中心、边缘消隐、冷色微光、少而干净的动态。
- 功能语言：遁入虚无、融于虚无、归零净化、伤害闪避、无相能量、自动修复。
- 避免：紫色混沌、星空空间、深渊生物、只套原版药水却写成高概念效果。

## 虚空

- 核心概念：空间、界域、距离折叠、仓储、传送、引力、裂隙。
- 视觉语言：深青、透明边界、空间裂缝、稀疏星点、宝库连接面。
- 功能语言：虚空宝库、虚空终端、传送石、磁吸、引力场、物流/传输。
- 避免：把虚空说成真正虚无；虚空是下位的空间属性，更偏实用设施。

## 混沌

- 核心概念：无序增幅、随机、暴走、扰动。
- 视觉语言：紫色、流动彩辉、不稳定辉光、高动态颜色。
- 功能语言：幸运扰动、战斗增幅、共鸣核心、高风险强化、商旅印。
- 避免：把混沌的紫色视觉套到无相工具/护甲上，除非该物品明确是混沌系。

## 深渊

- 核心概念：西幻魔性、灵魂、诅咒、复苏、深渊战意。
- 视觉语言：暗紫、魂火、骨纹/咒纹、更重的哥特轮廓。
- 功能语言：复苏、虹吸、魂锚、战号、战鼓、深渊生物和交易。
- 避免：把深渊厚重魔法感混进干净的虚无/无相装备。

## 当前归类

- 无相：工具、护甲、归零卷轴、净业铃、屏障核、守护印、相位/遁入类物品。
- 虚空：虚空宝库、虚空终端、传送石、磁核、引力符、停滞表、传输核心。
- 混沌：混沌血戒、幸运符、手甲、王冠、共鸣核、商旅印。
- 深渊：复苏宝珠、战号、魂瓶、魂锚、披肩、虹吸戒、深渊生物。

新增内容时先选属性，再让功能和贴图共同证明这个属性。若效果目前只是原版增益，就不要用过度文案包装成旗舰能力；要么诚实写清，要么补实现。
