# Temple of Nihility

虚无圣殿 — Minecraft 26.2 / NeoForge 26.2.0.7-beta 内容扩展模组。

当前仓库只保留正在维护的有效版本：NeoForge 26.2。旧版迁移备份、生成预览、本地测试输出和私有贴图生成脚本不会进入仓库。

## 功能

| 内容 | 说明 |
|------|------|
| 虚无材料 | 虚无碎片、粉尘、结晶、符文、核心、圣殿封印 |
| 虚无建筑 | 虚无石系方块、符文砖、虚无灯、虚无玻璃、虚无祭坛 |
| 武器工具 | 虚无剑、匕首、巨刃、镐、斧、铲、锄 |
| 装备 | 虚无头盔、胸甲、护腿、靴子 |
| 饰品 | Curios 可选联动：戒指、项链、腰带、披风、手甲、时沙计、魂锚等 |
| 虚无宝库 | 多方块仓储网络，每个宝库 243 格，相邻宝库自动合并 |
| 虚无终端 | 可放入 Curios 常见饰品栏，默认 `O` 打开绑定的宝库网络 |
| 常加载 | 空手潜行右键宝库可切换整个宝库网络常加载 |
| JEI 联动 | 终端界面支持 JEI 物品拖入搜索框、终端物品可被 JEI 点击识别 |
| 主动道具 | 灯核、复苏宝珠、遁影符、净化铃、洞察镜、屏障核、相位羽、引力符、战号 |
| 交易提示 | 虚无生物交易会明确提示材料不足、冷却和获得物品 |

## 玩家入口

- `/nihility help`：查看虚无宝库、虚无终端和常加载的简明说明。
- `/nihility terminal`：携带或佩戴已绑定的虚无终端时，打开绑定宝库网络。
- 拿着虚无终端潜行右键虚无宝库：绑定终端。
- 空手潜行右键虚无宝库：切换整个宝库网络常加载。

## 构建

需要 JDK 25。NeoForge 26.2 文档要求 Java 25，本项目使用 ModDevGradle。

```powershell
.\gradlew.bat build
```

构建产物：

```text
build/libs/temple-of-nihility-neoforge-26.2-1.0.0.jar
```

## 可选依赖

- Curios `15.0.0-beta.1+26.2`
- JEI `30.6.0.36` for NeoForge 26.2

当前本地工程使用测试实例中的 Curios / JEI jar 作为 `compileOnly` 依赖。若在新机器构建，需要先提供对应版本 jar，或把 `build.gradle` 改成可用 Maven 坐标。
